## Local development

```bash
# clone the repository
git clone git@github.com:7ingwang/hugothemes/uedog.git

# cd in the project directory
$ cd uedog/exampleSite/

# Start local dev server
$ hugo server --themesDir ../..
```
Follow the steps.
